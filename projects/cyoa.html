Cyoa: dark fantasy medieval adventure

The prompt for this project was to create a choose your own adventure game. Me and my teammate created a choose your own adventure about a dark fantasy medieval adventure. You wake up in the forest and hear neighing, you can run or investigate, prompting a different series of events and choices.

This project helped me practice collaboration and creativity. The most difficult part of this project was putting every file into the right place and then creating the links. we had two different folders for the two paths which made it easier to organize but a bit more difficult to link between them. We quickly figured out how to link between the folders and the rest of the project was easy. If we had more time id probably add another pathway with new options or add on to the current pathways, removing some current death endings and adding onto those.
